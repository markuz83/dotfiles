# qt6ct

Qt6 theming configuration for non-KDE/non-GNOME desktops.

## Dependencies

- `qt6ct`
- `breeze` (Qt style)
- `papirus-icon-theme`
- `MesloLGS Nerd Font Propo` (for Qt app fonts)

## Required environment variable

`QT_QPA_PLATFORMTHEME=qt6ct` must be set for Qt apps to pick up this config. This is set in `~/.config/systemd/user/dms.service.d/override.conf` for DMS sessions. On a new machine, verify it's exported before launching Qt apps.

## Notes

- `colors/matugen.conf` is auto-generated by DMS/matugen — don't edit it directly.
- `color_scheme_path=~/.config/qt6ct/colors/matugen.conf` — qt6ct supports `~/` paths.
- `style=Breeze` — requires the `breeze` package (part of KDE but installable standalone).
- `standard_dialogs=gtk3` — uses GTK3 file chooser dialogs instead of the Qt one, for consistency with the rest of the desktop.
