# helix

Helix editor configuration.

## Dependencies

- `helix`

## Notes

- `themes/matugen.toml` is auto-generated by matugen from `~/.config/matugen/templates/helix.toml`. Do not edit it directly — edit the template and re-run matugen.
- The `runtime/` directory contains helix's built-in grammars and queries and is intentionally not tracked (it's large and managed by the helix package).
