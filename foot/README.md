# foot

Foot terminal emulator config.

## Dependencies

- `foot`
- `JetBrainsMonoNL Nerd Font Mono` (AUR: `ttf-jetbrains-mono-nerd`)

## Notes

- `dank-colors.ini` is auto-generated by DMS/matugen — don't edit it directly.
- The `include=` directive uses `~/.config/foot/dank-colors.ini`. Foot supports `~/` paths but **not** other environment variables like `$XDG_CONFIG_HOME`.
- `alpha=0.4` in `[colors]` sets background transparency — requires compositor blur to look good.
- `selection-target=clipboard` means selected text goes to clipboard (not just primary). Change to `primary` if you prefer the traditional middle-click paste behavior.
- `[csd] preferred=none` disables client-side decorations — window borders come from the compositor.
