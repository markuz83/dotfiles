# mango

MangoWC Wayland compositor configuration.

## Dependencies

- `mangowc` (MangoWC compositor)
- DMS (DankMaterialShell)

## Important: most config lives in `dms/`

`config.conf` sources `./dms/colors.conf`, `./dms/outputs.conf`, `./dms/binds.conf`, etc. These are **auto-generated by DMS** and are not tracked here.

## Session startup

```
exec-once = dbus-update-activation-environment --systemd --all
exec-once = systemctl --user start mango-session.target
```

Like Hyprland, MangoWC propagates environment variables to systemd before starting the session target. The `mango-session.target` (defined in the `systemd` package) pulls in DMS.

## Animations

Animation curves follow Material Design 3 easing specs:
- Open: Emphasized `(0.2,0,0,1)` — draws attention on entrance
- Move: Standard `(0.2,0,0,1)` — natural repositioning
- Close: Accelerated `(0.4,0,1,1)` — snappy exit

## Colors

`rootcolor`, `maximizescreencolor`, etc. are set as hex `0xRRGGBBaa`. These are overridden by `dms/colors.conf` at runtime.
