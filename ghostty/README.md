# ghostty

Ghostty terminal emulator config.

## Dependencies

- `ghostty`
- `JetBrainsMono Nerd Font` (AUR: `ttf-jetbrains-mono-nerd`)

## Notes

- `themes/dankcolors` is auto-generated by DMS/matugen — don't edit it directly.
- `shaders/cursor_blaze.glsl` is a custom cursor shader. It's applied globally in the main `config` file.
- `config.bak` is a backup created by a previous config tool and is intentionally not tracked.
