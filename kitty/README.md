# kitty

Kitty terminal emulator configuration.

## Dependencies

- `kitty`
- `JetBrainsMono Nerd Font Mono` (AUR: `ttf-jetbrains-mono-nerd`)

## Notes

- `dank-theme.conf` and `dank-tabs.conf` are auto-generated by DMS/matugen — don't edit them directly. They are regenerated on every color change.
- `sessions/music.conf` is a kitty session file — opens a predefined window layout for music control.
- `background_opacity = 0.4` requires compositor blur.
- `hide_window_decorations yes` — window borders come from the compositor.
- `shell_integration enabled` allows kitty to track the current working directory and integrate with the shell prompt.
