# gtk

GTK 3 and GTK 4 theming configuration.

## Dependencies

- `adw-gtk3` — provides the base GTK 3/4 theme (symlinked from `/usr/share/themes/adw-gtk3/`)
- `papirus-icon-theme` — `Papirus-Dark` icon set

## What's tracked vs what's not

**Tracked (hand-written or matugen-generated):**
- `colors.css` / `dank-colors.css` — generated by matugen/DMS on each color change
- `gtk.css` — custom overrides
- `settings.ini` — GTK settings (theme, icon set, font, etc.)
- `assets/` — window button SVGs for the adw-gtk3 theme

**Not tracked:**
- `gtk-3.0/bookmarks` — personal file manager bookmarks
- `gtk-3.0/gamescope/` — gamescope-specific overrides
- `gtk-3.0/gtk.css.backup.*` — backup files created by GTK tools
- `gtk-4.0/gtk.css`, `gtk-4.0/gtk-dark.css`, `gtk-4.0/assets/` — symlinks to `/usr/share/themes/adw-gtk3/`; system-managed

## The backdrop fix

`dank-colors.css` gets a CSS block appended by `DankMaterialShell/scripts/sync-colors.sh` after every matugen run. This fixes Thunar icon label text contrast when the window is in an inactive (backdrop) state — without it, selected icon labels become unreadable. The script guards against duplication on repeated runs.

## gtk-4.0/gtk-4.0 directory

A nested `gtk-4.0/` directory exists inside `~/.config/gtk-4.0/` pointing to `adw-gtk3-dark`. This is a system symlink and is intentionally not tracked.
