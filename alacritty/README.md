# alacritty

Terminal emulator config.

## Dependencies

- `alacritty`
- `JetBrainsMono Nerd Font` (AUR: `ttf-jetbrains-mono-nerd`)
- A compositor with blur support (Hyprland, niri, MangoWC)

## Notes

- `dank-theme.toml` is auto-generated by DMS/matugen — don't edit it directly. Edit the matugen template instead.
- `opacity = 0.5` requires compositor blur to look right. Adjust to taste if running without blur.
- `TERM = "xterm-256color"` is set here rather than in `environment.d` since it's terminal-specific.
- `WINIT_X11_SCALE_FACTOR = "1.5"` is set for XWayland HiDPI scaling — remove or adjust if not on a HiDPI display.
- `live_config_reload = true` means changes apply without restarting.
