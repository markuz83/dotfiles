# rmpc

[rmpc](https://github.com/mierak/rmpc) — a TUI MPD client.

## Dependencies

- `rmpc`
- `mpd` (running, with socket at `~/.mpd/socket`)

## Notes

- `themes/matugen.ron` is auto-generated by matugen from `~/.config/matugen/templates/rmpc.ron`. Don't edit it directly — edit the template.
- Other themes (`catppuccin_mocha.ron`, `default.ron`) are included as alternatives.
- rmpc connects to MPD via the socket configured in its `config.ron`. Make sure this matches `bind_to_address` in `mpd.conf`.
