# superfile

[Superfile](https://github.com/yorukot/superfile) TUI file manager configuration.

## Dependencies

- `superfile` (AUR: `superfile-bin`)

## Notes

- `theme/` contains bundled community themes. The active theme is set in `config.toml`.
- `hotkeys.toml` overrides default keybindings.
- No integration with matugen/DMS — theme is selected manually.
